This is a position based fluid simulation for WebGL 2.0
It uses screen space fluid rendering for surface reconstruction
Additionally it supports mixing of multiple fluid colors and transparencies

Use the W, A, S and D keys to navigate through the scene
Go up and down with the Space Bar and the Control Key
To alter the view direction, click and drag with your mouse

Use the interface to control the simulation
To emit particles, press the "P" key

This simulation was developed for the Firefox Nightly Browser Version 45.0a1
Your can get it here: Nightly Download
You also need to enable WebGL 2.0
